# Screen Recorder App - A Simple and Efficient Way to Capture Your Screen on Ubuntu

## This lightweight screen recording application is designed to help you effortlessly capture your screen and save the footage on your system. Developed with Python 3.12, the app leverages a range of powerful libraries to ensure smooth performance and an intuitive user experience.

## Key Features:
1. Easy-to-Use Interface: Built with the user-friendly Tkinter library, the app provides an intuitive GUI with just two buttons: Start Recording and Stop Recording.
2. Seamless Screen Capture: Utilizing OpenCV and PyAutoGUI, the app captures your screen with precision, allowing you to record any activity or tutorial effortlessly.
3. Automatic File Naming: When you stop the recording, the file is saved automatically with a unique timestamp, ensuring that your recordings are properly organized and easy to find. Files are saved in the .avi format for high-quality video.
4. Efficient Performance: Thanks to Python's threading module, the app handles multiple processes simultaneously, allowing smooth, uninterrupted recordings.
5. Time Management: The app uses the datetime module to manage the current date and recording duration, providing an accurate timestamp for each session.
6. Built on a Stable Ubuntu Environment:
   - Python 3.12: The app is developed using the latest Python version to ensure optimal performance.
   - Virtual Environment: The app is designed to run in a dedicated Python virtual environment, making it easy to set up and use without any conflicts with system-wide packages.
7. Dependencies: 
   - OpenCV: For screen recording and video processing.
   - Numpy: To handle frame arrays efficiently.
   - PyAutoGUI: To capture screenshots of the screen.
   - Threading: To manage concurrent processes without performance hiccups.
   - 
Whether you're creating tutorials, recording gameplay, or simply capturing important on-screen information, this screen recorder app offers a simple and reliable solution for all your screen recording needs.
